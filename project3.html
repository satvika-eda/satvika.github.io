Biometric voting machine

In the present scenario, voting is done using ballot box which can be manipulative.Thus, the proposed system is to use fingerprint based voting.As fingerprint of an individual is unique, once a person has cast his/her vote the database will be updated and they cant cast their vote again in the same election.

There are various advantages of this: Cost effective, Low power consumption, Authenticated using fingerprint, Less manpower required, Avoids invalid voting, Time conscious in voting and counting, Ease of transportation


This project can be used for voting since it overcome all the draw backs of ordinary voting machine also provide additional security. Its main advantage is that since fingerprints of every person is unique and hence this system completely reduces the chance of invalid votes. The system can be manufactured simply as well as cheaply.
